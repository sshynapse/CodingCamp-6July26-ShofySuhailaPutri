## EXPENSE & BUDGET VISUALIZER

A simple web application for tracking daily expenses and visualizing spending by category.

## Live Demo
🔗 https://sshynapse.github.io/CodingCamp-6July26-ShofySuhailaPutri/

## Features
- Add transaction
- Delete transaction
- Search transaction
- Export CSV
- Dark Mode
- Local Storage
- Expense Pie Chart
- Responsive Design

Built with HTML, CSS, JavaScript, and Chart.js.
